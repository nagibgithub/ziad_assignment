Scope of var, let, and const

var

Variables declared with var have function scope.

let

Variables declared with let have block scope.

const

Variables declared with const have block scope and cannot be reassigned after initialization.

Use Cases of null and undefined

null

`null` is used to intentionally represent the absence of any object value or an empty variable.

  • Initializing variables or object properties before assigning a meaningful value.
  • As a placeholder for an object or value that doesn't exist yet but will be assigned later.
  • To indicate the absence of an element in an array or a missing property in an object.
let user = null; let book = { title: "JavaScript Programming", author: null };

undefined

`undefined` represents the absence of a value or an uninitialized variable.

  • Variables that have been declared but not assigned a value.
  • Function parameters that were not provided with values.
  • When an object property does not exist.

What do you mean by REST API?

A REST API (Representational State Transfer Application Programming Interface) is a set of rules and conventions for building and interacting with web services.

It follows the principles of simplicity, statelessness, and a client-server architecture to enable communication between different software systems over the internet.

REST APIs use standard HTTP methods like GET, POST, PUT, and DELETE to perform CRUD (Create, Read, Update, Delete) operations on resources, typically represented in formats like JSON or XML.

RESTful services are widely used for web and mobile applications to access and manipulate data from servers or databases in a standardized and scalable manner.
